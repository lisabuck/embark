# Embark & Co. — Website

Premium, design-led mobility equipment. Brand voice: "considered, not clinical."

## Structure
- `index.html` — the homepage (self-linked to images in `assets/`)
- `assets/` — all product photography (JPEGs)

## Run locally
Just open `index.html` in a browser, or for a local server:
`python3 -m http.server` then visit http://localhost:8000

## Deploy
Drag this whole folder onto vercel.com (Add New → Project), or connect this repo to Vercel for auto-deploy.

## Design system (do not drift)
- Palette: Bone #E0D7CC · Spanish Gray #A69A8C · Masala #5B5348 · Bistre #392418 · Drab Olive #21221B · Eerie Black #151613 · paper #f4efe8
- Type: Helvetica only ("Helvetica Neue", Helvetica, Arial, sans-serif). No web fonts, no serif.
- Flat colour fills only, no gradients on section grounds.

## Products (only these four)
1. Everyday Walker — pewter folding frame, canvas flap bag
2. Rover Everyday Walker — olive, front bag + side sling, hand brakes
3. Rover Home Walker — olive, matt tray top
4. Everyday Carry Walker — pewter frame, suspended soft bin

Shared spec (concept platform): Height 790–920mm · Width 610mm · Length 500mm · Weight 3.8kg · Castors 100mm · Wheels 125mm · Max user 160kg.
